# LD4LT Website (archive)

Linked Data for Language Technology is a W3C Community and Business Group investigating the intersection between Linked Data, Natural Language Processing and Knowledge Processing.

As of 2026, the CG is about to be dissolved, as much of our activity went into other forums, most notably the [European network for Web-centred linguistic data science (NexusLinguarum, 2019-2022)](https://nexuslinguarum.eu/) and the [Global Network on Large-Scale, Cross-domain and Multilingual Open Knowledge Graphs (GOBLIN, 2024-2028)](https://goblin-cost.eu/). However, these are temporary networks, and more activity may be expected after 2028.

To preserve previous work and to prepare to its re-initialization, we provide a mirror of the original website/wiki from https://www.w3.org/community/ld4lt. Note that we only keep the surface HTML.

To update the mirror from the web (and push the content back into this repo), run `make update`.
